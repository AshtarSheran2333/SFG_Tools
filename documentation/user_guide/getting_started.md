# Getting Started

This guide walks you through the basic setup and workflow
of the SFG_Tools.

---

## Requirements

You will need:

- Intel Fortran Compiler in your path (ifx)
- CMake version 3.1 and above
- Standard UNIX utilities (bash, make, etc.)

---

## Installation

1. make sure the [Requirements](./getting_started.md/#requirements) are met.
2. Clone and navigate to your repository:
```bash
git clone git@github.com:AshtarSheran2333/SFG_Tools.git
cd SFG_Tools
./compile.sh
```
3. The binaries should now be in SFG_Tools/bin/
